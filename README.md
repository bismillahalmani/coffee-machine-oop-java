# ☕ Coffee Machine Simulation (Java OOP Project)

## 📌 Overview
This is a Coffee Machine Simulation built using Java and Object-Oriented Programming (2nd Semester Project). It simulates a real-world coffee vending machine where users can order drinks, choose toppings, make payments, and receive receipts. It also includes an admin panel for machine status and maintenance.

---

## 🎯 Features
- Drink selection system  
- Topping options  
- Payment handling  
- Receipt generation  
- Order history using file handling  
- Admin features (machine status & refill system)  

---

## 🧠 Concepts Used
- Classes and Objects  
- Encapsulation  
- Abstraction  
- Modular Programming  
- File Handling  

---

## 🏗️ System Flow
1. User selects a drink  
2. Chooses topping  
3. System checks resources  
4. Payment is processed  
5. Coffee is prepared  
6. Receipt is generated  
7. Order is saved in file  

---

## 🚀 How to Run
1. Open project in any Java IDE (IntelliJ / Eclipse / VS Code)  
2. Compile all `.java` files inside `src/` folder  
3. Run `Main.java`  
4. Follow the console menu instructions
